**Marshall-Store-Quick-Proof**
===========================

A web-based tool for generating proof images with custom overlays and rounded corners. Users can upload screenshots, which are processed on an HTML canvas with layered images (background, watermark, header/footer, etc.). The final proof can be downloaded as a PNG file. Marshall-Store-Quick-Proof is designed to simplify the process of creating proof images for Marshall Store's marketing and sales teams.
